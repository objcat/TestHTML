• 1.标签选择器

    直接使用id进行选择 会选择到页面上所有的div

    <style> div { width: 100px; height: 100px; background-color: red; } </style> <div>我是标签选择器选择的div</div>

    例如:

    我是标签选择器选择的div

  • 2.id选择器

    直接使用id进行选择 因为id是唯一的 所以只会选择到一个div

    <style> #testDiv { width: 100px; height: 100px; background-color: green; } </style> <div id="testDiv">我是id选择器选择的div</div>

    例如:

    我是id选择器选择的div

  • 3.类选择器

    直接使用class进行选择 会选择到页面上所有的div

    <style> .testClass { width: 100px; height: 100px; background-color: black; } </style> <div class="testClass">我是class选择器选择的div</div>

    例如:

    我是class选择器选择的div

  • 4.后代选择器

    父标签可以选择自己包括所有标签 子标签 孙子标签 重孙子标签 ....

    <style> div span { background-color: green; } </style> <div style="width: 100px;height: 100px;background-color: red">我是父div <br> <span>我是子span</span> </div>

    例如:

    我是父div
    我是子span

  • 5.子元素选择器

    只能选择自己的儿子 不能选择孙子

    <style> div span { background-color: black; } </style> <div style="width: 200px;height: 200px;background-color: red">我是父div <br> <span>我是子span</span> <div style="width: 100px;height: 100px;background-color: yellow">我是子div <span>我是孙子span</span> </div> </div>

    例如:

    我是父div
    我是子span
    我是子div 我是孙子span
