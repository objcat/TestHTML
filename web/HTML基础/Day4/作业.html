  1. 列举出CSS常见的文本属性
    1. color 字体颜色
      值 描述
      <td style="color: red">红色的文本</td> 红色的文本
    2. line-height 行间距
      值 描述
      <td style="line-height: 100px">行间距 有没有发现我比被人高很多</td> 行间距 有没有发现我比被人高很多
    3. text-indent 文本缩进
      值 描述
      <td style="text-indent: 100px">文本缩进</td> 文本缩进 有没有发现我缩进了
    4. text-decoration 文字修饰
      值 描述
      none 默认。定义标准的文本。
      underline 定义文本下的一条线。
      overline 定义文本上的一条线。
      line-through 定义穿过文本下的一条线。
      blink 定义闪烁的文本。
      inherit 规定应该从父元素继承 text-decoration 属性的值。
    5. text-transform 属性控制文本的大小写
      值 描述
      none 默认。定义带有小写字母和大写字母的标准的文本。aaa
      uppercase 文本中的每个单词以大写字母开头。aaa
      lowercase 定义仅有大写字母。aaa
      line-through 定义无大写字母，仅有小写字母。aaa
      inherit 规定应该从父元素继承 text-transform 属性的值。aaa
    6. white-space 处理空格
      值 描述
      normal 默认。空白会被浏览器忽略。a b c d
      pre 空白会被浏览器保留。其行为方式类似 HTML 中的 pre 标签。a b c d
      nowrap 文本不会换行，文本会在在同一行上继续，直到遇到 br 标签为止。a b c d
      pre-wrap 保留空白符序列，但是正常地进行换行。a b c d pre 标签。
      pre-line 合并空白符序列，但是保留换行符。a b c d
      inherit 规定应该从父元素继承 white-space 属性的值。a b c d
